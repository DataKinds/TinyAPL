pattern: 'r←[x](F«G»H)y'
name: Fork
---

Note that `«` and `»` are two separate operators, but only meant to be used together.

If `x` is not provided, `r` is `(F y) G (H y)`; if `x` is provided, `r` is `(x F y) G (x H y)`.
